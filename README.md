# python-chess-engine

A fully functional desktop chess application built from scratch.

## Tech Stack
* **Language:** Python
* **GUI:** PyGame

## Key Features
* Algorithms for move validation.
* Check, checkmate, and stalemate detection.
* Support for special moves (castling, en passant, promotion).
* Theme support (By pressing "R")

## How to Run Locally
1. Clone the repository.
2. Navigate to the project folder: `cd src`
3. Start the game: `python main.py`
